# swapenv

Swap enviroment with breeze.

## Quick Start

- install `go install github.com/reduan2660/swapenv@latest` (binary coming soon)
- specify the enviroment in `.dev.env`, `.stage.env`, ... 
- run `swapenv load` to load the enviroments (all of them will disapper from your current directory)
- run `swapenv to <enviroment-name>` to swpa enviroments. e.g.: `swapenv to dev`
