# swapenv

Swap environment with breeze.

## Quick Start

- install `go install github.com/reduan2660/swapenv@latest` (binary coming soon)
- specify the environment in `.dev.env`, `.stage.env`, ...
- run `swapenv load` to load the environments (all of them will disappear from your current directory)
- run `swapenv to <environment-name>` to swpa environments. e.g.: `swapenv to dev`
